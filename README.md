# Titanic Neural Network Tutorial (Webpage)

## Overview
This repository contains the tutorial webpage for the Titanic neural network project.  
It explains preprocessing, model design, training, and evaluation, and embeds plots generated by the Python code.

## Contents
- `tutorial.html` – Webpage tutorial (<2000 words).
- `figures/` – PNG plots:
  - `titanic_nn_training_accuracy.png`
  - `titanic_nn_training_loss.png`
  - `titanic_nn_confusion_matrix.png`
  - `titanic_nn_roc_curve.png`
  - `titanic_nn_precision_recall.png`
- `requirements.txt` – Python dependencies (if needed for regenerating plots).

## How to Use
Open `tutorial.html` in your browser. Ensure the PNG figures are in the same folder so they display correctly.  
If you want to regenerate the plots, install the dependencies and run the notebook from Repo 3 (`titanic-nn-code`).
